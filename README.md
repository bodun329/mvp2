# Pokémon Explorer MVP

A simple and flashy React Single Page Application (SPA) that lets users explore Pokémon, view details, and manage favorites. Built as a Minimum Viable Product (MVP) for frontend development practice using React, Vite, and React Router.

## 🔹 Features

- Home page with navigation
- Pokémon List page displaying Pokémon fetched from the [PokéAPI](https://pokeapi.co/)
- Pokémon Detail page with:
  - Pokémon name, image, height, and weight
  - Add/remove Pokémon from favorites
- Responsive design for mobile, tablet, and desktop
- Flashy UI with:
  - Gradient background
  - Hover effects on navbar links and Pokémon cards
  - Styled buttons with hover transitions
- Basic state management using React Context API
- Routing using React Router (at least 4 routes)

---

## 🔹 Technologies Used

- React 18  
- Vite 4 (React template)  
- React Router DOM  
- PokéAPI for Pokémon data  
- Context API for global state management  
- CSS for styling and hover animations
